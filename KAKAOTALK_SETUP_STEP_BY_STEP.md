# 카카오톡 근로계약서 전송 설정 가이드

## 현재 상태
- ✅ 카카오톡 개발자 앱 등록 완료: "카페드로잉" (앱 ID: 1362883)
- ⚠️ 비즈 앱 미등록 상태

## 단계별 설정 방법

### 1단계: 카카오톡 메시지 제품 활성화

1. **카카오 개발자 콘솔 접속**
   - https://developers.kakao.com/
   - "카페드로잉" 앱 선택

2. **제품 설정 → 카카오톡 메시지**
   - 왼쪽 사이드바에서 "제품 설정" → "카카오톡 메시지" 클릭
   - "사용 설정"을 **ON**으로 변경
   - 저장

### 2단계: REST API 키 확인

1. **앱 설정 → 앱 키**
   - REST API 키 복사 (예: `c5d583e5ff16d309db38add5aa9032a4`)
   - 이 키를 환경변수에 추가해야 합니다

### 3단계: 비즈 앱 전환 (선택사항이지만 권장)

**알림톡을 사용하려면 비즈 앱이 필요합니다.**

1. **대시보드에서 "비즈 앱 등록" 버튼 클릭**
2. **사업자 정보 입력:**
   - 사업자 번호 (있는 경우)
   - 또는 개인 개발자인 경우: 본인인증 + 카카오비즈니스 통합 서비스 약관 동의
3. **비즈 앱 전환 완료**

### 4단계: 알림톡 템플릿 등록 (비즈 앱 전환 후)

1. **카카오 비즈니스 관리자 페이지 접속**
   - https://business.kakao.com/
   - 카카오톡 비즈니스 계정으로 로그인

2. **알림톡 템플릿 생성**
   - 템플릿 이름: "근로계약서 작성 완료 알림"
   - 메시지 내용 예시:
     ```
     [카페드로잉]
     안녕하세요, #{employeeName}님.
     
     근로계약서가 작성되었습니다.
     아래 링크에서 확인하세요.
     
     #{contractUrl}
     ```
   - 변수: `#{employeeName}`, `#{contractUrl}`

3. **템플릿 심사 신청 및 승인 대기**
   - 보통 1-2일 소요

### 5단계: 환경변수 설정

**Vercel 환경변수에 추가:**

1. **Vercel 대시보드 접속**
   - https://vercel.com/dashboard
   - `portal` 프로젝트 선택

2. **Settings → Environment Variables**

3. **다음 변수 추가:**
   ```
   KAKAO_REST_API_KEY=c5d583e5ff16d309db38add5aa9032a4
   ```
   (실제 REST API 키로 변경)

4. **Redeploy** 실행

### 6단계: API 구현

제가 코드를 작성해드리겠습니다. 다음 정보가 필요합니다:
- REST API 키 (앱 키에서 확인)
- 알림톡 템플릿 ID (템플릿 등록 후 발급)

## 대안: 비즈 앱 없이 사용하는 방법

비즈 앱 전환이 어려운 경우:

### 옵션 1: 친구톡 (카카오톡 채널 연결 필요)
- 카카오톡 채널 생성 및 연결
- 친구톡 API 사용
- 무료 (월 1,000건까지)

### 옵션 2: 카카오톡 링크 공유 (클라이언트 사이드)
- 사용자가 직접 링크를 공유
- 서버 사이드 자동 전송 불가

### 옵션 3: SMS 대체
- 네이버 클라우드 플랫폼 SMS API
- 또는 다른 SMS 서비스 사용
