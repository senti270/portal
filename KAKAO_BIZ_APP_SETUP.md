# 카카오 비즈 앱 전환 후 설정 가이드

## 현재 상태
- ✅ 비즈 앱 전환 심사 신청 완료
- ⏳ 심사 대기 중 (2-3일 소요 예정)

## 심사 완료 후 할 일

### 1단계: 비즈 앱 전환 확인
1. 카카오 개발자 콘솔 접속
2. "카페드로잉" 앱 선택
3. 대시보드에서 비즈 앱 상태 확인
4. "비즈 앱"으로 표시되면 전환 완료

### 2단계: 카카오 비즈니스 관리자 페이지 접속
1. https://business.kakao.com/ 접속
2. 카카오톡 비즈니스 계정으로 로그인
3. "카페드로잉" 앱과 연결 확인

### 3단계: 알림톡 템플릿 등록

#### 3-1. 템플릿 생성
1. **알림톡** 메뉴 선택
2. **템플릿 관리** → **템플릿 만들기** 클릭
3. **템플릿 정보 입력:**
   - 템플릿 이름: `근로계약서 작성 완료 알림`
   - 카테고리: `알림` 또는 `안내`
   - 발송용도: `근로계약서 작성 완료 안내`

#### 3-2. 메시지 내용 작성
```
[카페드로잉]
안녕하세요, #{employeeName}님.

근로계약서가 작성되었습니다.
아래 링크에서 확인하세요.

#{contractUrl}

※ 본 메시지는 근로계약서 작성 완료 안내입니다.
```

**변수 설정:**
- `#{employeeName}`: 근로자 이름
- `#{contractUrl}`: 계약서 PDF 다운로드 링크

#### 3-3. 버튼 추가 (선택사항)
- 버튼 타입: `웹링크`
- 버튼 이름: `계약서 확인하기`
- 링크: `#{contractUrl}`

#### 3-4. 템플릿 심사 신청
1. **저장** 클릭
2. **심사 신청** 클릭
3. 심사 완료 대기 (보통 1-2일)

### 4단계: 템플릿 ID 확인
1. 심사 승인 후 **템플릿 관리** 페이지로 이동
2. 등록한 템플릿 클릭
3. **템플릿 ID** 복사 (예: `12345`)
4. 이 ID를 환경변수에 추가하거나 코드에 입력

### 5단계: 환경변수 추가
Vercel 환경변수에 추가:
```
KAKAO_TALK_TEMPLATE_ID=템플릿_ID_여기에_입력
```

### 6단계: API 코드 활성화
제가 준비해둔 알림톡 API 코드가 자동으로 작동합니다.

## 참고사항

### 알림톡 사용 제한
- 유료 서비스 (건당 과금)
- 템플릿 메시지만 가능 (자유 형식 불가)
- 템플릿 심사 통과 필요

### 비용
- 알림톡 건당 과금 (카카오 비즈니스 요금제 확인)
- 월 사용량에 따라 할인 가능

### 개인정보 보호
- 전화번호는 암호화하여 전송
- 근로자 동의 필요 (카카오톡 수신 동의)
