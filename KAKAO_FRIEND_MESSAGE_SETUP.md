# 카카오톡 알림톡 설정 가이드

## ⚠️ 중요: 알림톡 vs 카카오톡 메시지

**근로계약서 전송에는 알림톡을 사용합니다!**

- **알림톡**: 비즈니스용 템플릿 메시지, 전화번호로 직접 발송, 자동 발송 가능 ✅
- **카카오톡 메시지 (친구톡)**: 친구 추가 필요, 수동 발송, 자동화 어려움 ❌

**"카카오톡 친구 목록 / 메시지" 기능은 신청하지 않아도 됩니다!**

## 현재 상태
- ✅ 비즈 앱 전환 승인 완료
- ⏳ 알림톡 템플릿 등록 필요

## 단계별 설정 방법

### 1단계: 알림톡 템플릿 등록 (카카오 비즈니스 관리자 페이지)

#### 1-1. 카카오 비즈니스 관리자 페이지 접속
1. https://business.kakao.com/ 접속
2. 카카오톡 비즈니스 계정으로 로그인
3. "카페드로잉" 앱과 연결 확인

#### 1-2. 알림톡 템플릿 생성
1. **알림톡** 메뉴 선택
2. **템플릿 관리** → **템플릿 만들기** 클릭
3. **템플릿 정보 입력:**
   - 템플릿 이름: `근로계약서 작성 완료 알림`
   - 카테고리: `알림` 또는 `안내`
   - 발송용도: `근로계약서 작성 완료 안내`

#### 1-3. 메시지 내용 작성
```
[카페드로잉]
안녕하세요, #{employeeName}님.

근로계약서가 작성되었습니다.
아래 링크에서 확인하세요.

#{contractUrl}

※ 본 메시지는 근로계약서 작성 완료 안내입니다.
```

**변수 설정:**
- `#{employeeName}`: 근로자 이름
- `#{contractUrl}`: 계약서 PDF 다운로드 링크

#### 1-4. 버튼 추가 (선택사항)
- 버튼 타입: `웹링크`
- 버튼 이름: `계약서 확인하기`
- 링크: `#{contractUrl}`

#### 1-5. 템플릿 심사 신청
1. **저장** 클릭
2. **심사 신청** 클릭
3. 심사 완료 대기 (보통 1-2일)

### 2단계: 템플릿 ID 확인 및 환경변수 설정

1. **템플릿 ID 확인**
   - 심사 승인 후 **템플릿 관리** 페이지로 이동
   - 등록한 템플릿 클릭
   - **템플릿 ID** 복사 (예: `12345`)

2. **Vercel 환경변수 추가**
   - Vercel 대시보드 → 프로젝트 → Settings → Environment Variables
   - 다음 변수 추가:
     ```
     KAKAO_TALK_TEMPLATE_ID=템플릿_ID_여기에_입력
     ```

### 3단계: API 코드 활성화

템플릿 ID를 알려주시면 제가 API 코드를 최종 활성화해드리겠습니다!

## 참고사항

### 알림톡 vs 카카오톡 메시지 (친구톡)
- **알림톡**: 템플릿 메시지, 심사 필요, 유료, 전화번호로 직접 발송 ✅
- **친구톡**: 자유 형식 메시지, 친구 추가 필요, 무료, 수동 발송 ❌

### 근로계약서 전송에는 알림톡 사용
- 템플릿으로 일관된 형식 유지
- 파일 링크 포함 가능
- 자동 발송 가능
- 친구 추가 불필요
