# 카카오톡 메시지 vs 알림톡 비교 가이드

## 두 가지 서비스의 차이

### 1. 카카오톡 메시지 (친구톡)
**특징:**
- 사용자가 **카카오톡 친구에게 직접 메시지**를 전송하는 기능
- 친구 추가가 필요함
- 자유 형식 메시지 가능
- 무료 (일일 쿼터 제한 있음)
- "카카오톡 친구 목록 / 메시지" 기능 신청 필요

**사용 시나리오:**
- 사용자가 앱에서 친구에게 메시지 전송
- 사용자가 직접 친구를 선택해서 메시지 보내기

**제한사항:**
- 친구 추가 필수
- 사용자가 직접 친구를 선택해야 함
- 자동 발송 어려움

---

### 2. 알림톡 (AlimTalk)
**특징:**
- **비즈니스용 템플릿 메시지** 전송
- 전화번호로 직접 발송 가능 (친구 추가 불필요)
- 템플릿 심사 필요
- 유료 (건당 과금)
- 비즈 앱 전환 완료 필요

**사용 시나리오:**
- 근로계약서 작성 완료 알림
- 주문 확인 알림
- 예약 확인 알림
- 기타 비즈니스 알림

**장점:**
- 전화번호만 있으면 발송 가능
- 자동 발송 가능
- 일관된 형식 (템플릿)
- 파일 링크 포함 가능

---

## 근로계약서 전송에는 어떤 것이 적합한가?

### ✅ **알림톡 추천**

**이유:**
1. **자동 발송 가능**: 근로계약서 작성 완료 시 자동으로 전송
2. **친구 추가 불필요**: 전화번호만 있으면 발송 가능
3. **템플릿으로 일관성**: 모든 근로자에게 동일한 형식으로 전송
4. **파일 링크 포함**: 계약서 PDF 링크를 안전하게 전송

**단점:**
- 유료 (건당 과금)
- 템플릿 심사 필요 (1-2일)

---

### ❌ 카카오톡 메시지 (친구톡)는 부적합

**이유:**
1. **친구 추가 필요**: 각 근로자를 친구로 추가해야 함
2. **수동 작업**: 사용자가 직접 친구를 선택해서 보내야 함
3. **자동화 어려움**: 근로계약서 작성 완료 시 자동 발송 불가능

---

## 현재 구현 상태

현재 코드는 **알림톡**을 기준으로 구현되어 있습니다:
- `app/api/employment-contract/send-kakao/route.ts`
- 알림톡 템플릿 ID 사용
- 전화번호로 직접 발송

---

## 결론

**근로계약서 전송에는 알림톡을 사용하세요!**

### 필요한 작업:
1. ✅ 비즈 앱 전환 완료 (이미 완료)
2. ⏳ 알림톡 템플릿 등록 (카카오 비즈니스 관리자 페이지)
3. ⏳ 템플릿 ID 환경변수 추가
4. ⏳ API 코드 활성화

### "카카오톡 친구 목록 / 메시지"는 필요 없습니다
- 알림톡을 사용하므로 이 기능은 신청하지 않아도 됩니다
- 다만, 향후 다른 기능(예: 사용자가 친구에게 공유하기)이 필요하면 신청 가능
