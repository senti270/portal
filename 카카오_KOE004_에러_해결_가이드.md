# 카카오톡 로그인 KOE004 에러 해결 가이드

## 📋 에러 내용

**KOE004: 앱 관리자 설정 오류**

카카오 개발자 콘솔에서 앱 관리자가 설정되지 않아 발생하는 오류입니다.

## 🔧 해결 방법

### 1. 카카오 개발자 콘솔 접속

1. [카카오 개발자 콘솔](https://developers.kakao.com/) 접속
2. 로그인
3. 내 애플리케이션 → 해당 앱 선택

### 2. 앱 관리자 설정

1. **앱 설정** → **앱 관리** 메뉴 클릭
2. **앱 관리자** 섹션 확인
3. 앱 관리자 이메일이 등록되어 있는지 확인
   - 없으면: 카카오 계정 이메일 입력 후 저장
   - 있으면: 이메일이 올바른지 확인

### 3. 카카오 로그인 활성화 확인

1. **제품 설정** → **카카오 로그인** 메뉴 클릭
2. **활성화 설정**이 "ON"인지 확인
3. **Redirect URI**가 올바르게 등록되어 있는지 확인:
   - `https://cdcdcd.kr`
   - `https://cdcdcd.kr/signup`
   - `http://localhost:3000` (개발용)
   - `http://localhost:3000/signup` (개발용)

### 4. 플랫폼 설정 확인

1. **앱 설정** → **플랫폼** 메뉴 클릭
2. **Web 플랫폼**이 등록되어 있는지 확인
3. 사이트 도메인이 올바르게 등록되어 있는지 확인:
   - `cdcdcd.kr`
   - `localhost` (개발용)

### 5. 앱 키 확인

1. **앱 설정** → **앱 키** 메뉴 클릭
2. **REST API 키** 확인
3. 환경 변수 `NEXT_PUBLIC_KAKAO_APP_KEY`와 일치하는지 확인

### 6. 카카오 로그인 동의항목 설정

1. **제품 설정** → **카카오 로그인** → **동의항목** 메뉴 클릭
2. 다음 동의항목이 활성화되어 있는지 확인:
   - 닉네임 (필수)
   - 프로필 사진 (선택)
   - 카카오계정(이메일) (선택)

## ⚠️ 중요 사항

1. **앱 관리자 설정은 필수입니다**
   - 앱 관리자 이메일이 없으면 KOE004 에러 발생
   - 반드시 카카오 계정 이메일을 등록해야 합니다

2. **변경 사항 반영 시간**
   - 설정 변경 후 최대 5분 정도 소요될 수 있습니다
   - 브라우저 캐시를 삭제한 후 다시 시도해보세요

3. **개발 환경 확인**
   - 로컬 개발 시에도 카카오 개발자 콘솔 설정이 필요합니다
   - localhost를 플랫폼 및 Redirect URI에 등록해야 합니다

## 🔍 추가 확인 사항

### 환경 변수 확인

Vercel 또는 로컬 환경에서 다음 환경 변수가 설정되어 있는지 확인:

```env
NEXT_PUBLIC_KAKAO_APP_KEY=your_rest_api_key
NEXT_PUBLIC_APP_URL=https://cdcdcd.kr
```

### 브라우저 콘솔 확인

브라우저 개발자 도구(F12) → Console 탭에서 다음 메시지 확인:
- `✅ 카카오 SDK 초기화 완료` - 정상
- `⚠️ NEXT_PUBLIC_KAKAO_APP_KEY가 설정되지 않았습니다.` - 환경 변수 확인 필요

## 📞 문제가 해결되지 않는 경우

1. 카카오 개발자 센터 문의: [고객센터](https://developers.kakao.com/support)
2. 앱 상태 확인: 카카오 개발자 콘솔에서 앱이 정상 상태인지 확인
3. 앱 키 재생성: 문제가 지속되면 앱 키를 재생성하고 환경 변수 업데이트
